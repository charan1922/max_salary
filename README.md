# max_salary of employess
 To find maximum salary from employess


## Build Setup

``` bash
clone the repo

run  cd max_salary

# install dependencies
npm install

# serve with hot reload at localhost:7000
npm run dev


```


## To display salaries from database
goto : (http://localhost:7000/salary)
